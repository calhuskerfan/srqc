# SR Queue Consumer

This project is the source code for the project described here [srqc](https://calhuskerfan.github.io/srqc)

## Running

Pre-requisites
- git
- dotnet9
- VSCode or Visual Studio

Steps
- sync this repo
- from a command prompt
    ```ps1
    dotnet run --project src/srqc -c Release
    ```
